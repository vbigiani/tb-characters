Introduction

This mod is a collection of characters packs for Baldur's Gate 2 (bundles of portraits and sound sets). As of now, all of these are ripped straight from Valve games.
If you enjoyed any of the soundsets, you are largely encouraged to buy the related games, both because they're excellent games on their own, and because this way you're giving something back after infringing on Valve's copyright.

Installation

Windows

The Bigg Character Pack is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a tb#characters folder, and Setup-tb#characters.exe in your BG2 folder. To install, simply double-click Setup-tb#character.exe and follow the instructions on screen.

Please re-run Setup-tb#character.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Macintosh

If the mod was previously installed, uninstall it before extracting the new version. Please note that this mod is incompatible with Mod Organizer. The Bigg Character Pack is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG2 folder. If properly extracted, you should have a tb#characters folder, Setup-tb#characters, and Setup-tb#characters.command in your BG2 folder. To install, simply double-click Setup-tb#characters.command and follow the instructions on screen.

Linux

Download the latest version of WeiDU for Linux from weidu.org and extract (at least) the WeiDU, weinstall and tolower executables in a $PATH directory (E.G. /usr/bin).
Following that, open a terminal, cd to your BG2 installation directory, run `tolower' and answer Y to both queries.
You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name.
If you're unsure, running tolower and choosing both options is the safe bet.
After that, run 'weinstall tb#characters' from your BG2 installation directory to install or uninstall the mod.

Compatibility notes

This mod should compatible with pretty much any and all mods ever made.
In particular, if the 'Soundset Subtitles' tweak of Ascension64's TobEx mod is installed, the soundsets will correctly have subtitles. This mod must be installed AFTER version BETA 0008 or earlier of TobEx. Current versions of TobEx can be installed in any relative order.
